# Blocks
块有可能是一个空的声明和语句序列，块是大括号匹配的。

```
Block = "{" StatementList "}" .
StatementList = { Statement ";" } .
```

代码中除了有显示的块外，还有隐式块：
 1. 宇宙块包括了所有的go源代码文本。
 2. 每一个包拥有一个包含所有属于该包的go源文本的包块。
 3. 每一个文件拥有一个包含所有属于该文件的go源文本的文件块。
 4. 每一个"if", "for"和"switch"语句是在它自己的隐式块中。
 5. 每一个在"switch"或者"select"中的子句表现为一个隐式的块。
